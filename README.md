# Team13_A.O.T._
**University of Missouri Senior Capstone Project**

## Project Overview
An AI Virtual Executive Assistant with a visual embodiment. This project integrates large language models, character personality engines, workflow automation, and 3D rednering to create a visually interactive assitant.

## Notable Software
Our system architecture relies on the following core components:
* **vLLM:** High-throughput anf memory-efficient local LLM inference.
* **SillyTavern:** User interface and persona prompt management.
* **n8n:** Workflow automation and webhook handling for system integration.
* **Unity:** 3D environment rendering and visual execution.
* **VRoid SDK:** Avatar modeling and integration within the Unity environment.
